Help: Glossary

Description

The "Glossary" module is a lexicon multi-categories.
Tested with versions 2.5.9 to 2.5.12.

This module uses the framework "janus" (see the chapter "installation")< br>

Main features:
- application of a color scheme for each category
- import of data from the "Lexikon" module which is no longer maintained.
- it can be cloned without difficulty.
