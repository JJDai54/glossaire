Exportzation

The "Export" tab allows you to export a glossary to import it on another site for example or to simply make a backup.
The export is compressed in "ZIP" format
The tables are in "YML" format

Export Options
Select a Category
 You can only export one category at a time.
Include Images
 Yes: Includes logos and definition images.
No: Allows you to reduce the archive size if images are not important
Include files
 Yes: Includes attached files in the archive.
No: Attached files are not included. Allows you to lighten the archive.
