Categories

To access the "Category" tab, you need at least one category, and assign it global permissions in the "Permissions" tab.

Category options - Table: glossary_categories
cat_id int Identifier
 Unique identifier automatically incremented
cat_name
 varchar(255) Category name
 String
cat_description text Description
 It will be displayed on the user side below the category selection.
cat_weight int Weight
 Order in which categories will be displayed
cat_logo varchar(80) Logo
 Logo or image associated with the category. It is displayed in the header of the lists.
cat_alphabarre varchar(255) Alphabet
 List of letters or alphanumeric characters displayed at the head and tail of the list to facilitate searches.
A click on one of the characters selects the entries that begin with it.
Special characters that can be inserted in this list v:
*   (asterisk) :  Selects all entries
#   (Hash mark): Selects entries that begin with a number
|    (Vertical bar): Inserts a line break. Allows you to separate letters and numbers, for example.
cat_alphabarre_mode tinyint Display mode
 yes: displays all characters in the bar, even those for which there are no entries.
no: displays only characters in the bar for which there are entries.
cat_upload_folder varchar(255) Files folder
 Name of the folder in which the logo, images and files associated with the entries will be stored.
This is a subfolder of the "/uploads" folder of the site.
Important: This name must be unique.
cat_colors_set varchar(50) Color set
 Allows you to select a predefined color set in the "JJD_Framework" framework
cat_is_acronym tinyint Acronym
 Defines whether the entry is an acronym.
If so, the first uppercase characters of each word in the short description will be bolded.
example:
Council of the Wise
will become
Conseil des Sages
Characters in the short definition preceded by the "/" character will also be bolded
cat_replace_arobase varchar(5)
String that replaces "@" in emails.
Example: [@]
email@domain.fr will be replaced by email[@]domain.fr and will block the display of the email application.
cat_pid int
 unused
 this field is scheduled to disappear in a future version
cat_userpager tinyint Number of lines per page
 Maximum number of lines displayed per page. It can be different for each category.
If zero, displays all entries in the category. To be used when there are few definitions in the category.
cat_br_after_term tinyint Line break
 no: The name and the short definition will be on the same line
yes: the short description will be put on the line
example no:
CdS: council of the Wise
example yes:
CdS
Council of the Wise
cat_show_terms_index tinyint
unused

cat_count_entries int Number of clicks
 Number of definitions of the category.
cat_active tinyint
Overrides permissions.
yes: The category will be displayed on the user side according to the permissions granted.
no: The category will not be displayed on the user side regardless of the permissions.
cat_date_format varchar(30) Date format
 Format of creation and update dates. day.
See the PHP documentation for more information.
cat_date_creation datetime(6) Creation date
 This date is automatically updated when the category is created
cat_date_update datetime(6) Update date
 This date is updated each time the category is modified.
