Glossaries / Entries

Definition options - glossary_entries table
ent_id int(8) Identifier
 Unique identifier automatically incremented
ent_cat_id int(10) Category
 Category ID .
It is possible to change the category of the entry
ent_creator varchar(80) Author
 Author Name
nt_term varchar(255) Term
 Main term or phrase , this field is required
ent_initiale varchar(5) Initial
 Automatically populated, allows for faster selection when clicking on the alphanumeric selection bar.
ent_shortdef varchar(255) Short definition
 String displayed after the term or on the line according to the category settings.
ent_is_acronym tinyint(1) Acronym
 Forces the default value defined in the category settings.
defined if the entry is an acronym.
If yes, the first uppercase characters of each word of the short description will be bolded.
example:
Council of the wise
will become
Conseil des Sages
The characters in the short definition preceded by the "/" character will also be bolded
ent_image varchar(255) Logo
 Logo or image displayed in the header of the definition.
ent_definition text Definition
 Full text of the entry.
Important: you must define the text editor in the module settings.
ent_reference text Reference
 Free field that allows you to provide additional information
ent_file_name varchar(80) Displayed name of the file
 Displayed name of the file attached if it exists. Not used at the moment, planned for a future version.
ent_file_path varchar(80) Physical name of the file
 Name of the file in the folder "/uploads/glossaire/categorie/"
ent_urls text Reference links
 Allows you to add as many links as you want on reference sites.
Enter one URL per line without a separator.
Each line can be composed of the title and the URL separated by a '|' without a space.
Example: Xoops France | https://www.frxoops.org/
The anchor will be automatically generated.
ent_email varchar(80) Email
 Court riel of a contact
ent_counter int(10) Counter
 Counts them click on the entries.
Important: this value should be taken with caution.
It is just an indication and may not be exact since it is possible to see the description without clicking on the name.
ent_status tinyint Status
 Inactive: The definition will not be displayed to the user
Proposed: definition proposed by a user. It is not visible until it has been approved.
Approved: The definition is visible to the user.
ent_flag int(10) Flag
 Field used during import to keep definition and category IDs consistent.
ent_date_creation datetime(6) Date of creation
 Field automatically updated - See display format in category settings.
ent_date_update datetime(6) Date updated
 Field automatically updated - See display format in category settings.
