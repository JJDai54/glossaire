Import

The import form allows:
  • Import from an external export made with the "Glossary" module or one of its clones.
  • For large files The import can be done from an export made with the "Glossary" module or one of its clones and placed in the "/upload/glossary/import" folder via FTP.
  • Direct import from the "Lexikon" module. This module is no longer maintained.


Import Options
Category
 Select the category to import the new definitions into.
"New Category" will add a number after the category name and folder name, to avoid duplicates.
The name and folder can be changed after import.
