Importation

Le formulaire d'importation permet :
  • L'importation à partir d'un export externes réalisé avec le module "Glossaire" ou un de ses clones.
  • Pour les gros fichier L'importation  peut se faire à partir d'un export réalisé avec le module "Glossaire" ou un de ses clones et placé dans le dossier "/upload/glossaire/import" via FTP.
  • L'importation directe à partir du module "Lexikon". Ce module n'est plus maintenu.


Options d'importation
Catégorie
 Sélectionnez la catégorie dans laquelle importer les nouvelles définitions.
"Nouvelle catégorie" ajoutera un numéro après le nom de la catégorie et le nom du dossier, afin déviter des doublons.
Le nom et le dossier pourront être modifiés après importation.
