Aide : Glossaire

Description

Le module "Glossaire" est un lexique multi-catégories.
Testé avec les version 2.5.9  à 2.5.12.

Ce module utilise le framework  "janus" (voir le chapitre "installation")

Principales caractérisques :
- application d'un jeu de couleurs pour chaque catégorie
- importation des données du module "Lexikon" qui n'est plus maintenu.
- il peut être cloné sans difficulté.
