Exportzation

L'onglet "Export" permet d'exporter un glossaire pour l'importer sur un autre site par exemple ou pour simplement en faire une sauvegarde.
L'export est compressé au format "ZIP"
Les tables sont au format "YML"

Options d'exportation
Sélection d'une catégorie
 Il n'est possible d'exporter qu'une catégorie à la fois.
Inclure les images
 Oui : Inclut les logos et les images des définitions.
Non : Permet d'alléger l'archive si les images ne sont pas importantes
Inclure les fichiers
 Oui : Inclut dans l'archive les fichiers attachés.
Non : Les fichiers attachés ne sont pas inclut. Permet d'alléger l'archives.
