Catégories

Pour accéder à l'onglet "Catégorie" il faut au moins une catégorie, et lui affecter les autorisations globales dans l'onglet "Autorisations".

Options des catégories - Table : glossaire_categories
cat_id int Identifiant
 Identifiant unique incrémenté automatiquement
cat_name
 varchar(255) Nom de la catégorie
 Chaine de caractères
cat_description text Description
 Elle sera affichée côté utilisateur en dessous du choix de la catégorie.
cat_weight int Poids
 Ordre dans lequel seront affichées les catégories
cat_logo varchar(80) Logo
 Logo ou image associé à la catégorie. Il est affiché dans l'entête des listes.
cat_alphabarre varchar(255) Alphabet
 Liste de lettres ou de caractères alphanumériques affichés en tête et en queue de liste pour faciliter les recherches.
Un clique sur un des caractère sélectionne les entrèes qui commence par celui-ci.
Caractères spéciaux que l'on peut insérer dans cette liste v:
*   (astérisque) :  Sélectionne toutes les entrées
#   (Dièse) : Sélectionne les entrèes qui commence par un chiffre
|    (Barre verticale) : insère un retrour à la ligne. Permet de séparer par exemple les lettres et les chiffres.
cat_alphabarre_mode tinyint Mode d'affichage
 oui : affiche tous les caractères de la barre, même ceux pour lesquels il n'y a pas d'entrées.
non : Affiche seulement les caractères de la barre pour lesquels il existe des entrées.
cat_upload_folder varchar(255) Dossier des fichiers
 Nom du dossier dans lequel seront stockés les logo, images et fichers associés aux entrées.
C'est un sous dossier du dossier "/uploads" du site.
Important : Ce nom doit être unique.
cat_colors_set varchar(50) Jeu de couleurs
 Permet de sélectionner un jeu de couleurs prédéfini dans le framework "JJD_Framework"
cat_is_acronym tinyint Acronyme
 défini si l'entrée est un acronyme.
Si oui, les premiers caractères majuscules de chaque mots de la description courte seront mis en gras.
exemple :
Conseil des sages
deviendra
Conseil des Sages
Les caractères de la définition courte précédés par le caractère "/" seront également mis en gras
cat_replace_arobase varchar(5)
Chaine de caractère qui remplace les "@" dans les courriels.
Exemple : [@]
courriel@domaine.fr sera remplacé par courriel[@]domaine.fr et bloquera l'affichage de l'application de messagerie.
cat_pid int
 inutilisé
 ce champ est voué à disparaitre dans une version future
cat_userpager tinyint Nombre de ligne par page
 Nombre de lignes maximum affiché par page. Il peut être différent pour chaque catégorie.
Si la valeur est zéro, affiche toutes les entrées de la catégorie. A utiliser lorsqu'il y a peu de définitions dans la catégorie.
cat_br_after_term tinyint Retour à la ligne
 non : Le nom et la définition courte seront sur la même ligne
oui : la description courte sera mis à la ligne
exemple non :
CdS : consil des Sages
exemple oui :
CdS
Conseil des Sages
cat_show_terms_index tinyint
inutilisé
cat_show_bin SMALLINT champ binaire
 Chaque bit de ce champ défini les champ à afficher et les options particulières :
ID
Définition courte
Magnifier
Définition
Auteur
Image
Référence
Fichier joint
Voir aussi
Contact
Compteur de visites
Date de création
Date de mise à jour
Position en haut des boutons de gestion
Position en bas des boutons de gestion
cat_count_entries int Nombre de clique
 Nombre de définitions de la cétégorie.
cat_active tinyint
Prioritaire sur les autorisations.
oui : La catégorie sera affichée côté utilisateur selon les permissions accordées.
non : la catégorie ne sera pas affichée côté utilisateur quelque soit les permissions.
cat_date_format varchar(30) Format des dates
 Format des dates de création et de mise à jour.
Voir la documentation PHP pour plus d'information.
cat_date_creation datetime(6) Date de création
 Cette date est mise à jour automatiquement lors de la création de la catégorie
cat_date_update datetime(6) Date de mise à jour
 Cette date est mise à jour à chaque modification de la catégorie.
