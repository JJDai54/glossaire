Installation

Important : il faut tout d'abord installer le framework  https://github.com/JJDai54/JJD-Framework
  • Suivre la procédure standart d'installation des modules Xoops
  • Donnez les autorisations globales
  • Créez une ou plusieurs catégories
  • Donnez les autorisations pour chaque catégories
  • Ajoutez des entrées pour chaque catégorie.
