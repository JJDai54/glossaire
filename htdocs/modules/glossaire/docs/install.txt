Read Me First
=============

First copy "JJD-Frameworks" into the "Frameworks" folder:
https://github.com/JJDai54/JJD-Framework

The use of "highslide" in the "Frameworks" folder is recommended (version 5.0.0)
Copy into the "Frameworks" folder the "highslide" library available here:
http://highslide.com/
Folder name must be "highslide"

Install "glossary" as another XOOPS module

In the "glossary/testdata/french" folder there are two archives to test the imports and the different functionalities.
